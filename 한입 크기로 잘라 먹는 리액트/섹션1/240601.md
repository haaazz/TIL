### 1.4 변수와 상수

- 변수 선언과 초기화
  - 변수를 새롭게 생성하는 것을 변수를 선언한다고 함
  - 선언과 동시에 값을 부여 -> 초기 값 설정 -> 초기화
  - let 변수명 = 값;
  - 같은 변수명 재선언 불가능
  - 값 재할당 가능

- 상수 선언과 초기화
  - const 상수명 = 값;
  - 값의 재할당이 불가능함 -> 선언과 동시에 초기화 반드시 필요함
  
- 네이밍 규칙
  - $, _를 제외한 기호는 사용 불가함
  - 숫자로 시작할 수 없음
  - 예약어를 사용할 수 없음
